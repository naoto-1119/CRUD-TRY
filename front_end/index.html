Solo Api Project

This is a public github repository providing CRUD API to GET/POST/PATCH/DELETE for a pre-set database schema.

Data Base Schema

  • Family Table- Only has House Name of a Family
  • Parents - Only has Parent's Information with relation to Family Table (Many to One)
  • Child - Only has Children's Information with relation to Family and Parents Table (Many to One)
